# HANDOFF Mediloop

## Estado actual del proyecto

Proyecto web Mediloop con:
- Landing/login (`index.html`)
- Área alumno (`alumnos.html`)
- Área tutor (`tutores.html`)
- Backend Node/Express en `localhost:4000`
- Sesión por rol (`student` / `tutor`)
- Persistencia básica en archivo JSON (`backend/data/store.json`)

## Estructura principal

- `frontend/public/index.html` → Landing + login
- `frontend/public/alumnos.html` → Vista alumno
- `frontend/public/tutores.html` → Vista tutor
- `backend/src/server.js` → servidor Express
- `backend/src/routes.js` → auth + endpoints
- `backend/data/store.json` → datos persistentes demo

## Flujo funcional implementado

### Login
- Login por rol desde landing.
- Redirección:
  - student -> `alumnos.html`
  - tutor -> `tutores.html`
- Protección de rol en frontend.
- Logout funcional (limpia sesión local).

### Tutor
- Carga de dashboard tutor desde backend.
- Pendientes/confirmados de asistencia:
  - confirmar
  - rechazar
  - confirmar todos
  - exportar CSV
- Rúbrica:
  - validación de campos y puntuaciones
  - guardar evaluación
  - reporte con total de evaluaciones
- Comunidad:
  - publicar post
  - cargar posts
- Estados visuales de éxito/error/cargando en acciones.

### Alumno
- Carga de datos base desde backend.
- Secciones completas visuales (hospitales, progreso, apoyo, recursos, mapa, novedades, comunidad).
- CTAs conectados a secciones internas.
- Estados visuales para acciones demo (sin alertas bruscas).

## UX/UI añadida recientemente

En alumno y tutor:
- Barra lateral fija por secciones (scrollspy).
- Botón flotante “volver arriba”.
- Modo compacto/expandido de barra lateral.
- Persistencia de preferencias (`localStorage`):
  - compactado de barra
  - última sección visitada
- Controles de preferencias:
  - Atrás
  - Olvidar sección
  - Deshacer preferencias
- Botonera compacta con iconos y tooltips custom:
  - flecha/tooltip estilo Mediloop
  - delay suave en aparición
